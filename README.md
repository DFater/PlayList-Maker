## Экран поиска треков

На этом экране пользователь может искать треки по любому непустому набору слов поискового запроса. Результаты поиска
представляют собой список, содержащий название трека/исполнителя, длительность, обложку. При активной строчке поиска показывает историю поиска.

## Плеер

На данном экране получаем больше информации о треке. Так же есть возможность прослушать 30 секундный отрывок трека. Добавить трек в "избранное" и плейлист.

## Экран избранных треков

Пользователь может добавлять треки в "Избранное", чтобы иметь возможность быстро вернуться к заинтересовавшему его
произведению. Добавить трек в "избранное" (или удалить из "избранного") можно на экране "Плеер".

## Экран настроек приложения

На данном экране есть возможность переключить темную/светлую тему приложения, вне зависимости от системной настройки. Есть возможность поделиться приложением, написать в поддержку, и перейти на страничку пользовательского соглашения.

## Экран создания плейлиста

Пользователь может создать плейлист добавив любую обложку из альбома телефона. Добавить название и описание создаваемому альбому. 

## Просмотр деталей плейлиста

Просмотр и удаление добавленных в плейлист треков. Возможность поделиться плейлистом. Удаление альбома и редактирование названия и описания. 

## Общие требования

- Приложение должно поддерживать устройства, начиная с Android 10 (minSdkVersion = 29)

## Стэк технологий
- Jetpack Navigation Component, ViewModel, Okhttp, Retrofit2, Room, Coroutines Flow, Koin, LiveData, RecyclerView,SharedPreferences, BottomNavigationView, Fragment
